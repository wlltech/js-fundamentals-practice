Diseño de una flor con canva

Programa que diseña una flor de colores en canva

Explicación del código

  1. var pantalla = document.querySelector("canvas"); Esta línea busca en el documento HTML un elemento especial llamado "canvas" y guarda una referencia a ese elemento en la variable llamada "pantalla". Es como encontrar un lienzo en blanco en una habitación y recordar su ubicación para usarlo más tarde.
  2. var pincel = pantalla.getContext("2d"); Esta línea obtiene una herramienta especial llamada "pincel" que se utiliza para dibujar en el lienzo (canvas) que encontramos anteriormente. Es como tomar un pincel específico que está diseñado para trabajar en el lienzo.
  3. pincel.fillStyle = "lightgray"; Esta línea elige un color para el pincel. En este caso, se establece el color de relleno en "gris claro". Es como decidir usar un color gris claro para rellenar las formas que vamos a dibujar.
  4. pincel.fillRect(0, 0, 600, 400); Esta línea dibuja un rectángulo en el lienzo utilizando el pincel y el color de relleno especificados anteriormente. El rectángulo se dibuja en la posición (0, 0) y tiene un ancho de 600 y una altura de 400. Es como usar el pincel para pintar un rectángulo de cierto tamaño en el lienzo, comenzando desde la esquina superior izquierda.
  5. Función dibujarCirculo Acepta cuatro parámetros: x, y, radio y color. pincel.fillStyle = color; Establece el color de relleno del pincel. pincel.beginPath(); Método del objeto pincel; Inicia un nuevo camino de dibujo. pincel.arc(x, y, radio, 0, 2 * Math.PI); arc es otro método del objeto pincel que dibuja un arco; x y y representan las coordenadas del centro del arco en el lienzo. El punto (x, y) será el centro del círculo que se dibujará. Radio es el radio del círculo que queremos dibujar. Determina el tamaño del arco. 0 especifica el ángulo inicial del arco en radianes, en este caso, se usa 0 para que el arco comience desde la posición de las 3 en punto de un reloj. 2 * Math.PI especifica el ángulo final del arco en radianes. Math.PI es una constante en JavaScript que representa el valor de π (pi), que es la relación entre la circunferencia de un círculo y su diámetro, multiplicarlo por 2 nos da un arco completo (360 grados) alrededor del centro del círculo. pincel.fill(); Rellena el arco con el color de relleno establecido previamente. El método fill del pincel se utiliza para aplicar el color de relleno y completar la forma.
