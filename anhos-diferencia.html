Años de Diferencia

Programa que muestra cuántos años de diferencia tienes con alguien.

Temas abordados

  • Variables
  • Funciones básicas

Ciclo del programa

  1. Se declara la variable miEdad para ingresar una de las edades
  2. Se declara la variable edadAlguien para ingrar la edad de la otra persona
  3. Se declara la función calcularDiferenciaEdades para definir la operación matemática
  4. Se declara la función saltoLinea para agregar un salto de línea a la respuesta (opcional)
  5. Se declara la función imprimir para configurar un mensaje con el resultado en pantalla. Dentro de esta función, se llaman las funciones imprimir y calcularDiferenciaEdades
  6. Finalmente, se llama la función imprimir anteriormente definida, y se pasa por parámetro el mensaje que desea imprimirse
