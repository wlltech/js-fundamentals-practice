Calcular edad a partir del año de nacimiento

Programa que calcula la edad dado el año de nacimiento

Temas abordados

  • Variables
  • Funciones
  • Función prompt
  • Operaciones matemáticas
  • Función document.write

Ciclo del programa

  1. Se declara la función calcularAnho que recibe como parámetro el año y devuelve el calculo del año 2023 menos el año recibido por parámetro.
  2. Se declara la función imprimir que recibe como parámetro un texto e imprime ese texto.
  3. Se declara la variable nombre para capturar el nombre del usuario y asignarlo a esta variable.
  4. Se declara la variable anhoNacimiento para capturar el año del usuario mostrando en una ventana emergente un mensaje que devuelve el valor almacenado en la variable nombre y un texto con la instrucción para que el usuario digite su año de nacimiento
  5. Se declara la variable calculoAnho para llamar la función calculoAnho pasando por parámetro el valor de la nariable anhoNacimiento
  6. Se llama la función imprimir donde se envía por parámetro un mensaje concatenado a la variable calculoAnho para que retorne el resultado
